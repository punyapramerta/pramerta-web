## 8. Panduan Implementasi On-Page

### Formula Meta Title
```
Homepage:   [Brand] | [Value Prop] — [Lokasi]
Produk:     [Nama Produk] Indonesia | [Brand] — [Benefit/Brand Partner]
Blog:       [Keyword Utama]: [Judul Lengkap]
```
> Panjang ideal: **50–60 karakter** (terlihat penuh di SERP)

### Formula Meta Description
```
[Deskripsi layanan + keyword sekunder] + [lokasi] + [angka/stats jika ada] + [CTA]
```
> Panjang ideal: **140–155 karakter**

### Contoh Meta Tag per Halaman

| Halaman | Title (maks 60 char) | Description (maks 155 char) |
|---------|---------------------|----------------------------|
| Homepage | `PAS HVAC \| Solusi HVAC Industri — Surabaya` | `Authorized distributor FRIMEC & GREE. AHU, Chiller, VRF, PAC, Ducting untuk industri Indonesia. 500+ proyek. Konsultasi gratis!` |
| AHU | `Air Handling Unit Indonesia \| PAS HVAC` | `Distributor & kontraktor AHU industri Indonesia. FRIMEC authorized, custom 500–50.000 CFM, HEPA filter. Instalasi profesional. Konsultasi gratis!` |
| Chiller | `Chiller Industri Indonesia \| PAS HVAC GREE` | `Jual chiller industri GREE di Indonesia. Water-cooled & air-cooled, 30–500 TR. Instalasi, commissioning & maintenance. Minta penawaran!` |
| PAC | `Precision Air Conditioning \| PAS HVAC` | `PAC untuk data center, cleanroom & ruang kritis. Suhu stabil ±1°C, 24/7, N+1 redundancy. Konsultasi teknis gratis sekarang!` |
| Ducting | `Ducting HVAC Indonesia \| Sheet Metal & Textile` | `Fabrikasi ducting SMACNA: sheet metal BJLS/SUS/ALU & textile duct. Untuk pabrik, rumah sakit, gedung komersial. Hubungi kami!` |

### Heading Structure (H1–H3)
- **H1:** 1 per halaman, sertakan keyword primer secara alami
- **H2:** Keyword sekunder & topik sub-section
- **H3:** Long-tail, FAQ, use case spesifik

### Internal Linking Prioritas
```
Blog posts  →  Halaman produk (anchor text = keyword target)
Homepage    →  Semua halaman produk
Portfolio   →  Produk terkait proyek
FAQ         →  Halaman produk relevan
```

### URL Structure
```
✅  /products/air-handling-unit           (deskriptif)
✅  /blog/cara-memilih-chiller-industri
✅  /portfolio/proyek-hvac-surabaya
❌  /products/p1                           (tidak deskriptif)
```

## Catatan Penting

| Aspek | Panduan |
|-------|---------|
| **Bahasa utama** | Bahasa Indonesia — target audience lokal |
| **Technical terms** | Boleh pakai EN (AHU, VRF, PAC, HEPA, SMACNA) — sudah lazim di industri |
| **Keyword density** | 1–2% per halaman, natural — hindari keyword stuffing |
| **Positioning** | Hindari kata "murah" — fokus pada "terpercaya", "authorized", "berpengalaman" |
| **Tracking** | Pantau Google Search Console tiap 2 minggu — impressions & clicks per halaman |
| **Review** | Perbarui guideline ini setiap 6 bulan sesuai perkembangan SERP |
